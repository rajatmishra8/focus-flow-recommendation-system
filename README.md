# Focus-Flow: Human-Centred Web Application & Focus Recommendation System

## Project Overview
Focus-Flow is an AI-driven web application designed to improve student productivity, focus, and digital well-being through personalized recommendations based on behavioural analysis.

The system analyzes stress levels, distraction behaviour, screen-time patterns, and academic performance indicators to generate intelligent productivity recommendations.

## Key Features
- Personalized productivity recommendation system
- Student self-assessment module
- NLP-based sentiment analysis
- Stress and distraction behaviour analysis
- Screen-time behaviour tracking
- Pomodoro productivity timer
- Assessment tracking module
- User authentication system
- Dashboard-based analytics

## Problem Statement
Students often struggle with distraction, stress, and inefficient study habits due to excessive digital consumption and poor productivity management.

Focus-Flow addresses this challenge by providing intelligent, data-driven recommendations to help students improve focus and academic performance.

## Tech Stack
- Python
- Flask
- SQLite
- TextBlob
- NLP
- HTML
- CSS
- JavaScript

## Workflow
1. Student inputs behavioural and productivity-related information
2. System analyzes stress, distraction, and productivity indicators
3. NLP-based sentiment analysis processes textual feedback
4. Recommendation engine generates personalized productivity suggestions
5. Dashboard displays productivity insights and tracking information

## Outcome
Developed a functional AI-driven productivity support platform combining analytics, behavioural assessment, and recommendation logic for student-focused digital well-being.
